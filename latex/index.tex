\hypertarget{index_autotoc_md0}{}\doxysubsection{\texorpdfstring{Introduzione}{Introduzione}}\label{index_autotoc_md0}
Benvenuti nella documentazione del progetto {\bfseries{Rubrica Contatti}}. ~\newline
 Questo progetto Maven scritto in {\bfseries{Java}} è organizzato in diverse componenti, ciascuna con uno scopo ben definito.

\DoxyHorRuler{0}
\hypertarget{index_autotoc_md2}{}\doxysubsection{\texorpdfstring{Struttura del Progetto Maven}{Struttura del Progetto Maven}}\label{index_autotoc_md2}
Il progetto è suddiviso nelle seguenti cartelle principali\+:


\begin{DoxyItemize}
\item {\bfseries{{\ttfamily GUI}}}\+: Contiene le classi per l\textquotesingle{}interfaccia grafica utente. ~\newline

\item {\bfseries{{\ttfamily gestione\+Contatti}}}\+: Comprende le classi per la logica di gestione dei contatti. ~\newline

\item {\bfseries{{\ttfamily rubrica}}}\+: Gestisce l\textquotesingle{}intera rubrica dei contatti. ~\newline

\item {\bfseries{{\ttfamily exception}}}\+: Definisce le eccezioni personalizzate del progetto. ~\newline

\item {\bfseries{{\ttfamily validazione\+Contatti}}}\+: Contiene le classi per validare i dati dei contatti inseriti. ~\newline

\item {\bfseries{{\ttfamily prove}}}\+: Include classi di test o di prova utilizzate durante lo sviluppo.
\end{DoxyItemize}
